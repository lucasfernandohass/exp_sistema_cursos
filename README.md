## Sobre o Projeto

O **Aprenda+** é uma plataforma de cursos online desenvolvida para facilitar o aprendizado por meio de **trilhas educacionais estruturadas, acompanhamento de progresso e avaliações automáticas**.

O sistema permite que os alunos estudem de forma flexível, acessando conteúdos a qualquer momento e de qualquer lugar.

---

## Objetivo

Criar uma solução completa de ensino EAD que permita:

- Assistir aulas gravadas  
- Realizar atividades  
- Acompanhar progresso  
- Emitir certificados  

---

## Usuários do Sistema

| Tipo            | Descrição |
|-----------------|----------|
| Alunos     | Consumem os cursos |
| Professores | Respondem dúvidas |
| Admins       | Gerenciam o sistema |

---

## Funcionalidades

### Autenticação
- Cadastro e login  
- Verificação de e-mail  
- Recuperação de senha  

### Cursos
- Catálogo com busca   
- Cadastro e edição de cursos  
- Avaliação por estrelas   

###  Matrículas
- Inscrição em cursos  
- Cancelamento de matrícula  
- Controle de acesso  

### Conteúdo
- Aulas gravadas  
- Atividades e provas  
- Perguntas e dúvidas  

### Progresso
- Acompanhamento em tempo real  
- Boletim de notas  

### Pagamentos
- Cartão   
- Boleto   
- Pix   

### Certificados
- Geração automática  
- Validação de autenticidade  

### Extras
- Notificações  
- Atendimento ao usuário  
- Painel administrativo  

---

## Requisitos Técnicos

- Suporte a até 10.000 usuários simultâneos  
- Tempo de resposta ≤ 10s  
- Segurança:
  - HTTPS  
  - Criptografia de senha   
- Responsivo (Desktop + Mobile)    
- Logs de erro  
- Conformidade com LGPD  

---

## Regras de Negócio

- Usuário deve se cadastrar com dados válidos  
- Acesso ao curso apenas após pagamento  
- Progresso é registrado automaticamente  
- Certificado só é gerado após aprovação  
- Administradores gerenciam cursos
