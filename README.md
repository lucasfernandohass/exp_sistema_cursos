## Sobre o Projeto

O **Aprenda+** é uma plataforma de cursos online desenvolvida para facilitar o aprendizado por meio de **trilhas educacionais estruturadas, acompanhamento de progresso e avaliações automáticas**.

O sistema permite que os alunos estudem de forma flexível, acessando conteúdos a qualquer momento e de qualquer lugar.

---

## Objetivo

Criar uma solução completa de ensino EAD que permita:

- Assistir aulas gravadas  
- Realizar atividades  
- Acompanhar progresso  
- Emitir certificados  

---

## Usuários do Sistema

| Tipo            | Descrição |
|-----------------|----------|
| Alunos     | Consumem os cursos |
| Professores | Respondem dúvidas |
| Admins       | Gerenciam o sistema |
